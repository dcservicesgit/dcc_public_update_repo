Cluster Update Distribution Endpoint

This site is used by DC CORE clusters to retrieve versioned runtime artifacts and verify update integrity. It is not intended for manual downloads or interactive use.

Automated Update Flow

Cluster nodes periodically check the update hash below. When the value changes, nodes fetch the corresponding artifacts and verify them using SHA-256 checksums. 

GET /UPDATE_HASH.txt

Integrity & Verification

All distributed artifacts are accompanied by SHA-256 checksum files. Clients are expected to verify integrity prior to activation. 

*.zip.sha256

Usage Notice

This endpoint is part of internal infrastructure. File layout and contents may change without notice.
